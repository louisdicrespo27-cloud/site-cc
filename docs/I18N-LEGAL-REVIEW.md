# Revisão jurídica — páginas internacionais (EN/FR)

**Estado:** Pendente de validação jurídica humana.

Este documento lista pontos a confirmar por advogado responsável antes de considerar as traduções EN/FR definitivas para efeitos de conformidade e comunicação profissional.

## Metodologia

Para cada página, verificar:

- [ ] Terminologia jurídica adequada ao contexto português e ao público-alvo
- [ ] Coerência com a versão portuguesa (sem alargar ou restringir direitos/obrigações)
- [ ] Ausência de promessas de resultado, prazos garantidos ou disponibilidade imediata
- [ ] Formulação institucional de exercício individual (sem sociedade/equipa)
